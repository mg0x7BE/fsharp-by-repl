# Simple F# Exercises

A structured F# learning path — from absolute basics to real-world projects.

Each exercise (`NNN.fsx`) teaches a concept with runnable examples, then gives you a task to solve.
Solutions are in the corresponding `NNNa.fsx` files.

## Usage

Open `.fsx` files in JetBrains Rider. Highlight code, send to F# Interactive (`Ctrl` + `\`).

You can also ask [Claude](https://claude.ai) to generate the next exercises for you — see `CLAUDE.md` for instructions.
